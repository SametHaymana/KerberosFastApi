# Kerberos Server

Building Server.

```bash
python3 -m venv env
```

For linux

```bash
source env/bin/activate
```

For Windows

```powershell
\env\Scripts\activate
```

Installing depencies

```bash
pip install -r requirements.txt

```

Starting Mysql server in docker-compose

```bash
docker-compose up
```

Finaly Start Servers And test it

```bash
python main.py

```
